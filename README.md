# 👨🏻‍💻 Customer Behavior Analysis – End-to-End Data Analytics Project  

This project demonstrates a complete end-to-end Data Analytics workflow, built to simulate real-world business analysis.  

It covers data cleaning, SQL-based business analysis, dashboard creation, and insight generation — just like a professional Data Analyst would do in an organization.

---

## 📌 Project Overview  

The goal of this project is to analyze customer shopping behavior and uncover insights related to purchasing patterns, subscription trends, product performance, and discount impact.

The workflow includes:

### ✅ Data Cleaning & Preparation (Python)
- Handling missing values  
- Feature engineering  
- Data transformation  
- Exploratory Data Analysis  

### ✅ Business Analysis (SQL)
- Customer segmentation  
- Revenue analysis  
- Subscription behavior insights  
- Product ranking using window functions  

### ✅ Dashboard Creation (Power BI)
- Interactive KPI dashboard  
- Category & product insights  
- Customer behavior trends  
- Subscription analysis  

### ✅ Final Reporting & Insights
- Clear business recommendations  
- Data-driven decision making  

---

## 🛠️ Tools & Technologies Used  

- Python  
- Pandas

---

## 📂 Project Workflow  

### 1️⃣ Data Cleaning in Python
- Load dataset  
- Clean column names  
- Handle null values  
- Feature engineering (age groups, frequency conversion, etc.)  
- Export cleaned data to MySQL  

### 2️⃣ SQL Analysis
- Write business-driven queries  
- Use CASE statements for segmentation  
- Apply window functions (ROW_NUMBER, RANK, DENSE_RANK)  
- Perform conditional aggregation  

### 3️⃣ Power BI Dashboard
- Connect MySQL to Power BI  
- Create data model  
- Build interactive dashboard  
- Generate insights  

### 4️⃣ Insight Generation
- Identify loyal customers  
- Analyze discount effectiveness  
- Compare shipping performance  
- Understand subscription behavior
